# 可复用提示词模板 — 股票/行业分析

> 每个模板包含：用途、适用场景和提示词本身。

## 模板1：周期顶部估值回测
> *来源：VLCC提示词1-2*

**适用场景**：周期性行业（航运、大宗商品、半导体、房地产），历史周期顶部可为当前估值提供参考。

```
对[行业]的[公司A]和[公司B]进行周期顶部估值回测：

1. 数据口径：
   - 使用最新真实[资产]数量（如船队规模、产能、门店数）。
   - 若为混合组合，折算为[主要资产]当量，使用合理比率。
   - 考虑规模效应：较小单位的单位成本更高。
   - 所有历史市值通胀调整至[当前年份]美元。

2. 周期定位：
   - [年份1]：[周期类型描述]（如超级周期、需求驱动）
   - [年份2]：[周期类型描述]（如脉冲、供给冲击）
   - [当前时期]：[描述]——定位在[年份1]和[年份2]之间。
   - 取中间偏上合理每资产市值进行估值。

3. 计算内容：
   - 通胀调整后每[资产]当量市值（各周期顶部）。
   - 当前[资产]数量 × 当量系数。
   - 保守/中性/乐观三档目标市值。
   - 与当前市值对比计算上行空间。
   - 结论：谁弹性更大？谁性价比/确定性更高？

4. 结合当前看多论点增强分析：
   - [供给约束1]
   - [供给约束2]
   - [需求驱动]
   - 考虑当前价格是否将突破通胀调整后的历史高点。

5. 输出要求：简洁、可建模、无矛盾。
```

## 模板2：基本面深度研究（模型结果分歧）
> *来源：VLCC提示词6*

**适用场景**：初始模型显示意外结果（如两只类似股票隐含上行空间差异巨大）。

```
[公司A]和[公司B]的[时间段]价格走势非常相似（均[走势描述]）。
但我的模型显示[公司A]的上行空间远大于[公司B]，与市场表现不符。

深度研究两家公司：
1. 获取最新公开文件、财报和行业数据。
2. 对比：[关键指标]（如资产组合、负债水平、合同结构、管理质量、资本开支）。
3. 找出解释模型分歧的结构性差异。
4. 按可能性排序提出3-5个潜在解释。
5. 建议模型调整方向。
```

## 模板3：经营杠杆 / SaaS经济学分析
> *来源：VLCC提示词8（第1部分）*

**适用场景**：任何高固定成本行业，利润在盈亏平衡之上呈非线性增长。

```
分析[行业]的经营杠杆（类SaaS经济学）：

1. [资产类型]的利润类似SaaS——收入增长[X]%可导致利润指数级增长，
   因为[固定成本项]基本固定。

2. 对行业内每种资产类型：
   - [资产类型1]：计算盈亏平衡[费率/价格]、当前[费率/价格]，
     以及在1倍、1.5倍、2倍、3倍盈亏平衡时的利润倍增器。
   - [资产类型2]：同上。
   - [资产类型3]：同上。

3. 展示非线性利润曲线：
   - 盈亏平衡：利润$0
   - 2倍盈亏平衡：[X]%利润率
   - 3倍盈亏平衡：[Y]%利润率（展示加速效应）

4. 应用于[公司A]和[公司B]：
   - 当前运营费率 vs 盈亏平衡
   - 每$[单位]费率增长的增量利润
   - 哪家公司从经营杠杆中受益更多？为什么？
```

## 模板4：供给侧现实检验
> *来源：VLCC提示词8（第2部分——OPEC检验，通用化）*

**适用场景**：主要供给侧公告（OPEC、台积电资本开支、新房开工）需要与实际数据核实。

```
对[实体]宣布的[供给行动]进行现实检验：

1. 宣布 vs 实际：[实体]宣布了[具体行动]，但公告不一定兑现。检查：
   - 历史模式：宣布的[变化]多少比例成为实际？
   - 补偿性调整（如抵消性削减、延迟、取消）。
   - 公告与执行之间的"前置"差距。

2. 与历史周期对比：
   - [周期1]：[实体]实际做了什么 vs 宣布了什么？
   - [周期2]：同上。
   - 当前：实际情况是什么？

3. 从一手来源获取真实[产出/供给]数据。

4. 对[行业]投资论点的影响：现实是支持还是削弱当前投资逻辑？
```

## 模板5：合约/收入结构对比分析
> *来源：VLCC提示词12-13*

**适用场景**：对比收入锁定策略不同的公司（现货vs合约、固定vs浮动、对冲vs未对冲）。

```
分析[公司A]和[公司B]的合约/收入结构差异：

1. 收入结构：
   - [公司A]：[X]%现货 / [Y]%合约 / [Z]%对冲
   - [公司B]：[X]%现货 / [Y]%合约 / [Z]%对冲

2. 计算：
   - 对[价格/费率]变化的敏感性（每$[单位]变动）
   - 盈利弹性（费率每变动1%→盈利变动%）
   - 收入稳定性（季度盈利标准差）
   - 各合约结构下的盈亏平衡费率

3. 生成对比：
   - 费率敏感性图表（盈利 vs 费率情景）
   - 上行捕获：费率飙升时谁受益更大？
   - 下行保护：费率崩塌时谁更受保护？

4. 交叉验证：若其他来源显示不同的合约比例，需调和数据。区分：
   - 锁定率（近期已签约天数占比）
   - 结构性比例（长期现货/合约比率）
   - 这是两个不同的指标，经常被混淆。
```

## 模板6：跨市场同行比较（不同交易所）
> *来源：VLCC提示词15、16b*

**适用场景**：对比在不同交易所上市的公司（如A股vs纽交所），存在估值溢价/折价。

```
对比[公司A]（[交易所A]）与[公司B]（[交易所B]）：

1. 关键：两家公司均为多元化——不要用总市值÷[主要板块]数量。

2. 使用4种方法：
   a. 每总资产估值：市值÷总[资产]数（所有板块）。
   b. 分部加总（SOTP）：估算各板块价值，剥离[主要板块]进行同类比较。
   c. PE对比：相同假设下的远期PE。
   d. 隐藏价值：识别单一板块指标中不可见的资产。

3. 计算真实跨市场溢价/折价：
   - 原始溢价（误导性）：[X]倍
   - SOTP调整后溢价（正确）：[Y]倍
   - 用结构性因素解释残差

4. 结论：溢价是合理的、过度的、还是实际上是折价？
```

## 模板7：全组合盈利重算
> *来源：VLCC提示词16*

**适用场景**：初始盈利模型只覆盖了多元化公司的一个板块。

```
[公司]的[年份]盈利模型仅计算了[主要板块]的弹性。
但[公司]共有[N]个[资产]横跨[N]个板块：
- [板块1]：[数量]个单位，[费率/价格]特征
- [板块2]：[数量]个单位，[费率/价格]特征
- [板块N]：...

用当前市场费率分别重算各板块盈利。
展示：旧模型（单板块）vs 新模型（全组合）的差异和影响。
```

## 模板8：替代情景建模
> *来源：VLCC提示词17-18*

**适用场景**：当前市场状况与卖方共识显著分歧。

```
当前[指标]为[当前值]。卖方共识假设[共识值]。
建模替代情景：以[当前值]为真正的基准。

1. 替代基准的理由：[供给原因]、[需求原因]、[结构原因]
2. 并列对比两个基准下的：盈利、EPS、PE、目标价
3. 关键：更新报告所有章节（规则11），而非仅新增独立章节
4. 核心问题：卖方何时重新锚定至[替代值]？催化剂是什么？
```

## 模板9：Day1Global框架应用
> *来源：VLCC提示词7、19*

**适用场景**：每份股票分析。这是总体分析框架。

```
对[公司]应用Day1Global tech-earnings-deepdive框架：

6大投资视角：质量复利、想象力增长、基本面多空、深度价值、催化剂驱动、宏观战术
反偏见框架：6种认知陷阱的识别与应对
事前尸检："[时间]后你亏了40%。哪里出了错？"——3-5个情景含概率和预警信号
```
