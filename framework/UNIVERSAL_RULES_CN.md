# 通用股票分析规则

> **适用于AI辅助权益研究的行业通用规则。**
> 从VLCC航运分析19+轮迭代提示词中提炼。
> 适用于任何行业——在此基础上叠加行业特定规则即可。
> 最后更新：2026年3月4日。

---

## 🔴 工作流程（每次必须）

### 规则1：双语输出
每次报告变更必须同时应用于**英文和中文**（或其他目标语言）版本。永远不要只更新一个。
- 确保数字、表格和结论在各语言间**完全一致**。

### 规则2：提示词与决策日志
每次对话中引入新分析、修正或数据后：
- 更新提示词日志，记录提问内容、发现和关键决策。
- 在所有目标语言中维护日志。
- 每条记录包含日期和核心发现。

### 规则3：版本控制纪律
任何文件变更后，暂存→提交→推送。不留本地未推送的变更。
- 提交信息应总结变更内容和原因。
- 使用有意义的提交信息，而非"更新文件"。

### 规则4：数据交叉验证
发布任何数字前：
- 用简单估算验证敏感性数据。
- 合理性检查：`数量 × 费率 × 时间 × 利润率 × 税率 × 汇率 = 结果`
- 跨模型对比——差异超过20%时标注。
- 不同来源数据冲突时，明确标注差异。

## 🟡 分析标准

### 规则5：多模型共识方法
进行新研究时用3-5个模型运行分析：
- 对所有模型使用相同的、自包含的提示词。
- 汇编共识+标注分歧。
- 标注哪些模型使用了正确/错误的假设。
- 向正确解读数据的模型倾斜权重。

**建议模型组合**（2026年）：Claude Opus 4.6、Claude Sonnet 4.6、GPT-5.2、GPT-5.1、Gemini 3 Pro。
> *提示：对于可能返回空值的模型（如Gemini），在提示词中加入"不要使用工具"。*

### 规则6：经营杠杆 / SaaS经济学
适用于任何高固定成本行业（航运、航空、半导体、SaaS、电信）：
- 计算盈亏平衡点（利润转正的收入水平）。
- 展示不同收入/费率情景下的非线性利润倍增器。
- 论证超过盈亏平衡后，增量收入≈纯利润。
- 此框架适用于任何具有高固定成本和可变定价的业务。

### 规则7：目标价格为必填项
每份股票分析必须包含：
- 保守/基准/牛市三档情景（至少3个）。
- 12个月和24个月目标价。
- 明确的PE（或相关估值倍数）假设及理由。
- 相对当前价格的上行/下行百分比。
- 分情景投资建议（见规则11）。

### 规则8：报告结构
每份公司/对比分析应包含：
1. **核心结论/TL;DR** — 关键指标、评级、目标价
2. **公司概况** — 商业模式、业务板块、核心财务数据
3. **行业/市场背景** — 当前状况、历史对比
4. **盈利模型** — 分板块含敏感性表格
5. **经营杠杆分析** — 盈亏平衡、利润倍增器
6. **周期/估值框架** — 历史PE/PB压缩规律
7. **多模型共识** — 各模型目标价
8. **风险矩阵** — 含概率和影响程度
9. **投资建议** — 分情景目标价、配置、触发条件
10. **Day1Global框架** — 模块C/L/O、6大视角、反偏见、事前尸检
11. **跨市场同行比较** — 如适用

### 规则9：获取最新市场数据
进行任何估值工作前：
- 获取当前股价和市值（在波动性行业变化极快）。
- 获取所有相关业务板块的当前运营费率/价格。
- 不要在未验证的情况下复用旧数据。

### 规则10：全组合 / 多板块建模
分析多元化公司时：
- 分别建模**每个业务板块**——永远不要用总市值÷主营业务数量。
- 考虑不同板块的利润率、增速和周期性差异。
- 当板块构成不同时，使用分部加总法（SOTP）进行跨公司比较。

### 规则11：全文件情景一致性
新增情景或假设时（如不同的价格/费率基准）：
- **更新报告中所有引用相关指标的章节**——而非仅新增一个独立章节。
- 核心结论必须展示所有情景。
- 所有目标价表格必须展示多情景列。
- 投资建议必须包含分情景的建议、目标价和配置。
- 任何含PE、NI、EPS或目标价的章节必须反映所有建模情景。

> **原因**：独立的"4B节"不够，如果第九节仍只引用旧假设的话。

### 规则12：Day1Global框架应用
每份股票分析报告必须应用[Day1Global tech-earnings-deepdive](https://github.com/webleon/tech-earnings-deepdive-openclaw-skill)框架：

**必选模块：**
| 模块 | 名称 | 聚焦点 |
|---|---|---|
| A | 收入质量 | 收入驱动、结构、可持续性 |
| B | 盈利能力 | 利润率、经营杠杆、成本结构 |
| C | 现金流 | **"重中之重"** — FCF、FCF收益率、资本纪律 |
| D | 前瞻指引 | 管理层展望、PE压缩路径 |
| E | 竞争格局 | 同行、护城河、市场份额 |
| K | 估值矩阵 | 多情景目标价、EV/EBITDA、P/NAV |
| L | 股权与管理层 | 股东结构、利益一致性、关联交易风险 |
| O | 会计质量 | 折旧政策、减值风险、表外项目 |

**必选视角（6个）：**
1. **质量复利**（巴菲特/芒格）— 持久竞争优势？
2. **想象力增长**（Baillie Gifford/ARK）— 10倍期权？
3. **基本面多空**（Tiger Cubs）— 市场错误定价了什么？
4. **深度价值**（Klarman/Marks）— 安全边际？
5. **催化剂驱动**（Tepper/Ackman）— 具体的价值释放事件？
6. **宏观战术**（Druckenmiller）— 周期/宏观定位？

**必选框架：**
- **反偏见**：识别并应对锚定、简化、近因、确认、幸存者和叙事偏见。
- **事前尸检**："一年后你亏了40%。哪里出了错？"——列举3-5个具体情景，含概率和预警信号。
