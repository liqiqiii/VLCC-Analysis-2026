ℹ️ How to Use — Data Sources ▼

Enter daily observations from shipping reports to build a transit trend. Here's where to find data:

📰 Lloyd's List (Paywalled — Summaries Free)

  • Weekly tanker market report includes Hormuz transit counts
  • Search for "Hormuz" or "VLCC" at lloydslist.com

🌐 gCaptain.com (Free)

  • Regular coverage of Hormuz chokepoint disruptions
  • Check gcaptain.com — search "Hormuz" or "Strait" or "VLCC"

💹 Bloomberg Shipping Terminal

  • If you have access: BMAP function for vessel positions
  • Bloomberg tanker reports often cite Yanbu loading counts

🛰️ VesselFinder.com (Free Manual Check)

  • Go to vesselfinder.com
  • Zoom to Strait of Hormuz: approximately 26.5°N, 56.3°E
  • Filter by vessel type: Tanker > VLCC (200K+ DWT)
  • Count laden VLCCs transiting outbound (southwest direction)
  • Count VLCCs anchored in Gulf near the strait

📡 Signal Ocean

  • Professional AIS-based tanker tracking platform
  • Provides transit counts and waiting vessel analytics

💡 Tips

  • Enter data at the same time daily for consistency
  • Use "Notable Vessels" to track specific DHT/FRO/INSW ships
  • Export JSON regularly as backup to your GitHub repo
  • TD3C rate can be found on the Baltic Exchange or shipping news
